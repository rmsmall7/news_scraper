# News Scraper
New Scraper is a web application that will all the user to view and leave comments on news articles from the New York Times

Technologies Used
Front-end:
  * HTML5
  * JavaScript
  
Back-end
  * JavaScript(Node.js)
  * NPM Packages: body-parser, express, router, cheerio, axios
  * Handlebars
  * MongoDB
